# html5-advanced
A professional front-end started template

## Структура папок и файлов

```
├── dist/                        # Результат работы FrontEnd
├── src/                         # Исходные файлы проекта
│   ├── image/                  # Исходники картинок
│   │   └── favicon/           # Исходник для генерации Favicon'a
│   ├── template/               # Исходники шаблонизатора JADE
│   │   ├── include/
│   ├── sсss/                  # Исходники препроцессора SASS
│   │   ├── defaults/        
│   │   ├── layouts/        
│   │   ├── pages/  
│   │   ├── vendors/
│   │   └── _config.scss  
├── .csscomb.json              # Конфигурация форматирования CSS
├── .eslintrc                  # Конфигурация проверки JavaScript в ESLint
├── .editorconfig              # Конфигурация настроек редактора кода
├── .gitignore                 # Список исключённых файлов из Git
├── package.json               # Список модулей и прочей информации
└── readme.md                  # Документация шаблона
```
